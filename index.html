Developer Tools Directory

A curated list of core developer tools with links and short descriptions.

Core Code Editors & IDEs
  • Visual Studio Code — https://code.visualstudio.com — Free, fast, extensible editor with huge ecosystem.
  • IntelliJ IDEA — https://www.jetbrains.com/idea — Premier Java/Kotlin IDE with deep code intelligence.
  • WebStorm — https://www.jetbrains.com/webstorm — JavaScript/TypeScript IDE focused on web development.
  • PyCharm — https://www.jetbrains.com/pycharm — Python IDE with strong refactoring and debugging.
  • Atom (Archived) — https://github.com/atom/atom — Hackable editor; discontinued but still used.
  • Sublime Text — https://www.sublimetext.com — Lightweight, fast editor with powerful multi-cursor editing.
  • Vim — https://www.vim.org — Modal text editor for power users.
  • Neovim — https://neovim.io — Modern Vim with Lua scripting and better plugin architecture.
Version Control & Collaboration
  • Git — https://git-scm.com — Distributed version control system.
  • GitHub — https://github.com — Code hosting, issues, CI/CD, and collaboration.
  • GitLab — https://gitlab.com — Git hosting with integrated DevOps pipelines.
  • Bitbucket — https://bitbucket.org — Git hosting with strong Jira integration.
Build Tools & Package Managers

JavaScript / Web

  • npm — https://www.npmjs.com — Default Node.js package manager.
  • Yarn — https://yarnpkg.com — Faster, more deterministic installs.
  • pnpm — https://pnpm.io — Space-efficient package manager using symlinks.
  • Webpack — https://webpack.js.org — Powerful bundler for complex web apps.
  • Vite — https://vitejs.dev — Lightning-fast dev server and bundler.

Python

  • pip — https://pip.pypa.io — Standard Python package manager.
  • Poetry — https://python-poetry.org — Dependency management and virtual environments.

Java

  • Maven — https://maven.apache.org — XML-based build automation.
  • Gradle — https://gradle.org — Flexible build system using Groovy/Kotlin DSL.
Testing Frameworks

JavaScript

  • Jest — https://jestjs.io — Popular testing framework with snapshot support.
  • Mocha — https://mochajs.org — Flexible JavaScript test runner.
  • Cypress — https://www.cypress.io — Browser automation and end-to-end testing.

Python

  • pytest — https://pytest.org — Simple, powerful testing with fixtures.

Java

  • JUnit — https://junit.org — Standard Java unit testing framework.
API & Backend Tools
  • Postman — https://www.postman.com — API testing, collections, and automation.
  • Insomnia — https://insomnia.rest — REST/GraphQL client with a clean UI.
  • Swagger / OpenAPI — https://swagger.io — API documentation and interactive UI.
  • Docker — https://www.docker.com — Containerization for reproducible environments.
  • Kubernetes — https://kubernetes.io — Container orchestration at scale.
Databases & Tools

SQL

  • MySQL — https://www.mysql.com — Popular open-source relational database.
  • PostgreSQL — https://www.postgresql.org — Advanced SQL database with rich features.
  • SQLite — https://sqlite.org — Lightweight embedded database.

NoSQL

  • MongoDB — https://www.mongodb.com — Document-based NoSQL database.
  • Redis — https://redis.io — In-memory key-value store, great for caching.

GUI Clients

  • DBeaver — https://dbeaver.io — Universal database client.
  • TablePlus — https://tableplus.com — Fast, modern database GUI.
